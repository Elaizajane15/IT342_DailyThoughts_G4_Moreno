# Project Overview

## Structure

- `backend/` — Spring Boot application scaffold.
- `web/` — Frontend page to log “What I Did Right Now”.
- `README.md` — Activity log and next steps.
- `docs/` — Documentation.

## Frontend

- `web/index.html` loads `css/styles.css` and `js/app.js`.
- Saves entries to `localStorage` and renders a list with timestamps.

## Backend

- Basic Spring Boot project initialized; future APIs can be added.
